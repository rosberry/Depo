## Requirements

- Swift 5.3+

## Installation
### Makefile
```
git clone git@github.com:rosberry/Depo.git
cd Depo
make
```

## Documentation
```

OVERVIEW: Main

USAGE: depo <subcommand>

OPTIONS:
  --version               Show the version.
  -h, --help              Show help information.

SUBCOMMANDS:
  init
  update
  install (default)
  build
  pod
  carthage
  spm

  See 'depo help <subcommand>' for detailed help.
```
